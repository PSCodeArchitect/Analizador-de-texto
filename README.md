# Analizador de Texto

## Descripción
Este proyecto es un analizador de texto hecho en Python.
Permite analizar un texto desde un archivo o desde la consola.

## Funcionalidades
- Normalización de texto (minúsculas y sin signos de puntuación)
- Tokenización
- Conteo de palabras
- Tokens únicos
- Top 10 palabras más frecuentes
- Longitud promedio de palabras
- Consulta interactiva de palabras

## Cómo ejecutar el programa

1. tener Python 3 instalado.
2. Colocar el archivo `textanalyzer.py` en una carpeta.
3. Ejecutar en la terminal:


5. Seleccionar el modo deseado:
- 1 → Leer desde archivo
- 2 → Escribir texto manualmente (modo consola)
- 3 → Ejecutar pruebas

## Ejemplo de uso

Si eliges el modo consola, debes pegar el texto y escribir "END"en una nueva línea para finalizar la entrada.

Después de analizar el texto (ya sea por archivo o por consola), puedes escribir una palabra para ver sus estadísticas dentro del texto.  
Para salir del programa, escribe `exit`.
